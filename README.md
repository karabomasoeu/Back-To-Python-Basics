# Back-To-Python-Basics

This is a python based project.

There are a few problem sets to help with the process of learning as a beginner.

 1. Greet.py is an example of basic Hello World.
 2. Grade_Calculator.py is a program that calculates the grade of sudents results.
 3. Name_Input.py 
 4. Number_Input.py
 5. PosNegNumber.py
 6. Simple_Addtion.py this is basic addition.
 7. Store_and_Print.py
 8. String_Concatenation.py this shows how two strings can be printed as one joint string, a work or sentence.
 9. Temp_Conversion.py this program converts temptrature between Celsius and Fahrenheit.
